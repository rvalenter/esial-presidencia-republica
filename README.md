# Esial 2.0

## Funcionalidades:

- Gestão de usuários;
- Gestão de Acessos;
- Perfil do Usuário;
- Nota Técnica - Ministérios;
- Parlamento:
    - Contatos;
    - Perfil Parlamentar;
    - Estatísticas;
    - Proposições;
    - Votações;
    - Discursos;
    - Destaques; e
    - Relatório PDF.
- Relatórios:
    - Comissão; e
    - Parlamentar.
- Manual do Usuário.

## Tecnologias

- [Vue] - HTML enhanced for web apps!
- [TailwindCss] - great UI boilerplate for modern web apps
- [PHP/Laravel] - Php framwork.

## Installation

Dev:

- clone this project;
- npm install;
- composer install;
- migrate;
- run sail;
- access: localhost

Prod:

- run Kubernets.
